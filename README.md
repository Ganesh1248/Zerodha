# 💹 Zerodha Clone - Full Stack Trading Dashboard

This is a **full stack clone of Zerodha's trading platform UI**, built as a personal project to understand frontend design patterns and user workflows in fintech dashboards. The frontend replicates the clean and responsive design of Zerodha, styled using **Bootstrap**, and simulates trading functionalities using mock data.

---

## 🚀 Features

- 🔐 **User Authentication** (Login/Register UI only)
- 📊 **Dashboard** with account summary, holdings, and watchlists
- 📈 **Live Stock Cards UI** with real-time price mock (JS intervals)
- 🧾 **Order Placement UI** – Buy/Sell screens with form validation
- 📁 **Portfolio & Holdings Page** with sorting/filtering functionality
- 🔄 Clean Routing using **React Router**
- 🎨 **Responsive UI** using Bootstrap 5 (mobile-first)
- 📥 Realistic Zerodha UX replica for learning purpose
